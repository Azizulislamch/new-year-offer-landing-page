# 🎉 New Year Offer Landing Page

A modern, elegant, and fully responsive landing page designed for New Year promotions and holiday sales. Built with a focus on clean UI/UX and mobile-first responsiveness.

## 🚀 Live Demo
Check out the live project here:  
**https://azizulislamch.github.io/new-year-offer-landing-page/**

---

## ✨ Key Features
* **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, and Desktop).
* **Modern Design:** Uses a vibrant color palette (Red, White, and Gold) to match the festive theme.
* **Interactive Sections:**
    * Dynamic Hero Banner with festive typography.
    * Holiday Sale section featuring a 50% discount offer.
    * Gift Box and Celebration elements with smooth layouts.
    * Newsletter Subscription form for lead generation.
* **Pixel Perfect:** Carefully crafted based on modern design standards.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic markup for better SEO and structure.
* **CSS3:** Custom styling including Flexbox, Grid, and Media Queries.
* **Google Fonts:** Specialized typography (Merriweather and Inter).
* **Font Awesome:** For professional icons.
